Skills and Web Languages.

I'm proficient in HTML5, CSS3, Sass, Javascript, jQuery, C#, LINQ, SQL, Bootstrap, and Git.

Microsoft's .NET platform, or more specifically ASP .NET Core, is a framework for developing cross-platform web applications written in C#. Using this framework, I've built 3 applications using the Model-View-Controller(MVC) architectural pattern.

These are the big three when it comes to web development on the front-end. HTML is like the basic skeleton of a web page, then CSS adds the skin and features, and Javascript is the muscle handling the more dynamic elements. I know all of these well, including some of their lackeys such as the CSS preprocessor, Sass, and the Javascript library, jQuery.

SQL is a language designed for managing data held in a relational database management system (RDBMS), or for stream processing in a relational data stream management system (RDSMS). It is particularly useful in handling structured data, i.e. data incorporating relations among entities and variables. I've become well acquainted with employing SQL database as well as PostgreSql while developing my applications.

Bootstrap is the most widely used HTML, CSS, and Javascript library in the world. Their collection of code snippets and web elements streamline the development process and I owe them greatly for their clean and thorough documentation.

Git is a popular Version Control System(VCS) that allows users to save snap-shots of project files instead of just catalogging file-based changes. This approach opens up developers to a much safer and efficient development process. I used the collaborative Git based service GitHub to manage my projects through development as well as work with a team in group projects.

  • Project and ticket tracking with full CRUD functionality
  • Informative and downloadable data tables
